Install flow

Practical first steps

Desktop

Windows users can start with the installer, or use the portable zip if they want to run from a folder or flash drive.

Android

Use the old, new, emulator, or universal APK depending on the device. The universal build is the safest one to try first.

TV

Use the TV APK directly on the television, then use the remote to move between buttons and open the highlighted file.

Controls

On touch devices, tap the cards. On TV and desktop, use arrows or Tab to move between buttons and Enter or Select to open one.

Arrow keys move focus Enter / Select opens Backspace / Escape returns
